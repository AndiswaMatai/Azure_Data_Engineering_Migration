## Overview
End-to-end Azure solution for real-time fleet and financial reporting across distributed environments.

## Problem
Fragmented on-prem SQL Server systems caused delays in reporting and limited visibility into fleet performance.

## Solution
- Automated data ingestion using Azure Data Factory (ADF)
- Centralized storage in Azure Data Lake Storage (ADLS)
- Scalable transformation in Azure Databricks
- Analytics-ready datasets in Azure Synapse
- Executive dashboards in Power BI

## Impact
- Reporting turnaround reduced from days to under 2 hours
- Fleet utilization improved by 12%
- Enabled faster decision-making across operations and finance

## Tech Stack
- **Ingestion:** Azure Data Factory
- **Storage:** Azure Data Lake Storage
- **Transformation:** Azure Databricks, Python, SQL
- **Analytics:** Azure Synapse Analytics, Power BI
- **Security & Governance:** Azure AD, Key Vault
